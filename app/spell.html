Enter the text

Note

  • 1. Press the Start button to begin the practice.
  • 2. Type the word in the text, press Enter key or click the Check button, It will tell you the next word If the typed word is correct else you may have try again.
  • 3. Press the Repeat button, If you want to hear the word one more time. The maximum limit is 3.
  • 4. If you are not sure about the current word, You can skip word by pressing the Next button to get next word.
  • 5. Click the Clear button to reset the text box.
